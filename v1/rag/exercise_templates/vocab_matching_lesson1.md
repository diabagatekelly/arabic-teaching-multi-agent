# Vocabulary Matching Template

## Pattern
Match Arabic words with their English translations.

## Example Set 1
**Arabic Words:**
1. كِتَاب
2. مَدْرَسَة
3. قَلَم
4. طَالِب

**English Translations:**
A. pen
B. school
C. student
D. book

**Answers:** 1-D, 2-B, 3-A, 4-C

## Example Set 2
**Arabic Words:**
1. كَبِير
2. صَغِير
3. جَمِيل
4. جَدِيد

**English Translations:**
A. new
B. small
C. beautiful
D. big

**Answers:** 1-D, 2-B, 3-C, 4-A

## Generation Guidelines
- Use 4-6 words per set
- Include only Lesson 1 vocabulary
- Randomize order to prevent position-based guessing
- Can group by theme (nouns, adjectives, verbs)
- Provide transliterations when reviewing answers
