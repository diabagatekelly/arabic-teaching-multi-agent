# Antonym Matching Template

## Pattern
Match Arabic words with their opposites.

## Example Set 1
**Word:** كَبِير (kabiir - big)
**Opposite:** صَغِير (saghiir - small)

**Word:** جَدِيد (jadiid - new)
**Opposite:** قَدِيم (qadiim - old)

**Word:** جَمِيل (jamiil - beautiful)
**Opposite:** قَبِيح (qabiih - ugly)

**Word:** طَوِيل (tawiil - long/tall)
**Opposite:** قَصِير (qasiir - short)

## Exercise Format
**Match the opposites:**
1. كَبِير
2. جَدِيد
3. نَظِيف
4. طَوِيل

A) قَصِير (short)
B) صَغِير (small)
C) قَذِر (dirty)
D) قَدِيم (old)

**Answers:** 1-B, 2-D, 3-C, 4-A

## Generation Guidelines
- Use adjectives from Lesson 1
- Focus on common opposites
- Include transliterations
- Can present as matching or fill-in-blank
- Help build semantic relationships
