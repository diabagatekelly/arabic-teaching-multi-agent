# Visual Recognition Template

## Pattern
Label images or identify objects with Arabic vocabulary.

## Example 1: Classroom Objects
**[Image: Book]**
**Options:**
A) قَلَم (qalam - pen)
B) كِتَاب (kitaab - book) ✓
C) دَفْتَر (daftar - notebook)

## Example 2: Size Description
**[Image: Large building]**
**Description:** بَيْت ___ (bayt ___ - a ___ house)
**Options:**
A) صَغِير (saghiir - small)
B) كَبِير (kabiir - big) ✓

## Example 3: Multiple Items
**[Scene: Classroom with desk, chair, blackboard]**
**Identify and label:**
1. ___ (desk) → مِكْتَب (maktab)
2. ___ (chair) → كُرْسِي (kursii)
3. ___ (blackboard) → سَبُّورَة (sabbuura)

## Generation Guidelines
- Use simple, clear contexts
- Focus on concrete nouns from Lesson 1
- Can describe images instead of showing them
- Include both recognition and production
- Helpful for visual learners
