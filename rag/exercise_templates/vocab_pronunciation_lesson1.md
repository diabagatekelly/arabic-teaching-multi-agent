# Pronunciation Practice Template

## Pattern
Match Arabic script with correct transliteration/pronunciation.

## Example 1
**Arabic:** كِتَاب
**Options:**
A) kitaab ✓
B) kataab
C) kiitaab
D) kutaab

## Example 2
**Arabic:** مَدْرَسَة
**Options:**
A) madarasa
B) madrasa ✓
C) madraasah
D) mudrasa

## Example 3
**Arabic:** طَالِب
**Options:**
A) taalib ✓
B) taleb
C) taleeb
D) taaleb

## Example 4
**Arabic:** مُدَرِّس
**Options:**
A) muddaris
B) mudarris ✓
C) madaris
D) mudaras

## Generation Guidelines
- Include diacritic marks (tashkeel) in Arabic
- Provide plausible but incorrect pronunciation options
- Focus on distinguishing similar sounds
- Use vocabulary from Lesson 1
- Can also reverse: give transliteration, match to Arabic script
