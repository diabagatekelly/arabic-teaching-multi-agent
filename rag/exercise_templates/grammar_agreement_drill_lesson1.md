# Gender Agreement Drill Template

## Pattern
Quick-fire gender agreement practice - add ة to feminine forms.

## Drill Set 1: Adjective Agreement
**Masculine → Feminine**
1. كَبِير (kabiir) → كَبِيرَة (kabiira)
2. صَغِير (saghiir) → صَغِيرَة (saghiira)
3. جَمِيل (jamiil) → جَمِيلَة (jamiila)
4. جَدِيد (jadiid) → جَدِيدَة (jadiida)
5. قَدِيم (qadiim) → قَدِيمَة (qadiima)

## Drill Set 2: Match to Noun
**Choose correct adjective form:**
1. كِتَاب ___ (book - masculine) → جَدِيد ✓
2. مَدْرَسَة ___ (school - feminine) → كَبِيرَة ✓
3. قَلَم ___ (pen - masculine) → صَغِير ✓
4. طَاوِلَة ___ (table - feminine) → جَمِيلَة ✓
5. بَيْت ___ (house - masculine) → قَدِيم ✓

## Drill Set 3: Rapid Recognition
**Identify if gender agreement is correct:**
1. كِتَاب كَبِيرَة - ❌ (should be كَبِير)
2. مَدْرَسَة جَدِيدَة - ✓
3. قَلَم صَغِيرَة - ❌ (should be صَغِير)
4. طَاوِلَة جَمِيل - ❌ (should be جَمِيلَة)
5. بَيْت جَدِيد - ✓

## Generation Guidelines
- Focus on speed and automaticity
- Include 10-15 items per drill set
- Mix correct and incorrect for identification drills
- Use only Lesson 1 vocabulary
- Keep it fast-paced for skill building
- Emphasize the ة marker pattern
